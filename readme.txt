== Description ==

MediaPress addons that works with BuddyPress and allow user to mark their gallery and media as featured. SideAdmin can show user's featured media or gallery in header as well as sidebar.

== Installation ==

This section describes how to install the plugin and get it working.

You may need to follow the following steps.
1. Visit Dashboard->Plugin->Add New
2. Search for "mediaPress featured Content"
3. Click install and then activate.
4. Please visit Dashboard->Mediapress->Settings->Addons to configure it.

Or manually install it.
1. Download `mpp-featured-content.xyz.zip` (where xyz is version number )and extract
2. Upload `mpp-featured-content` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Please visit Dashboard->Mediapress->Settings->Addons to configure it.
