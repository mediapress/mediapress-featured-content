Let your users show their featured photos(or audio, video, doc) and featured galleries on their BuddyPress profile.

== Description ==

MediaPress Featured content plugin empowers BuddyPress users to showcase their featured photo or any supported media type and gallery type on their profile.
It also support showcasing featured media and gallery for BuddyPress groups.

**Only BudyPress user and group galleries can be featured**
==Features==
MediaPress Featured Content plugin for BuddyPress supports:

* Featuring user galleries or media
* Featuring group galleries or media
* Displaying featured media or gallery in BuddyPress user profile header and Group header.
* Widgets for displaying featured media/gallery
* Shortcodes for displaying featured media/gallery.
* And a lot more options.
* Please see the [MediaPress Featured Content](https://buddydev.com/plugins/mpp-featured-content/) page on BuddyDev for all features. 
* Localizable

We support the plugin via [BuddyDev support forums](https://buddydev.com/support/forums/).
Thank you for using it.

== Installation ==

1. Please download the zip file and extract it.
2.  Upload the `mpp-featured-content` to the `/wp-content/plugins/mpp-featured-content`directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the Dashboard->MediaPress->Settings->Addon tab to configure the plugin.
